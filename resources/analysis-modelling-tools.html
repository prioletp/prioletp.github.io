Analysis & Modelling Tools

📊

Tools for planning observations, visualising interferometric data, fitting models and reconstructing images from visibility measurements.

JMMC Software Suite

🎯
ASPRO 2
The Astronomical Software to PRepare Observations — plan VLTI and CHARA observations, visualise uv-coverage, and estimate visibility for simple models.
↗ 📊
OIFits Explorer
Interactive tool for browsing and plotting OIFITS files — quickly inspect visibilities, closure phases, uv-planes and spectral channels.
↗ 📐
LITpro
Model fitting software for interferometric data — fit geometric models (uniform disks, rings, Gaussians, binaries) to OI-FITS data using χ² minimisation.
↗ ⭐
SearchCal
Find interferometric calibrator stars — estimates angular diameters from photometry and provides calibrator lists for your science target.
↗

Image Reconstruction

🖼️
MiRA
Multi-aperture Image Reconstruction Algorithm — reconstruct images from optical interferometric data using regularised inverse methods.
↗ 🖼️
SQUEEZE
Image reconstruction using MCMC and simulated annealing — well suited for sparse interferometric datasets with limited uv-coverage.
↗ 🖼️
IRBis
Interferometric image reconstruction using the bispectrum — a robust approach for image recovery from closure phases and visibilities.
↗

Model Fitting & Analysis

🎲
emcee
The MCMC Hammer — an affine-invariant ensemble sampler widely used for Bayesian parameter estimation in astrophysics.
↗ 🎲
dynesty
Dynamic nested sampling package for Bayesian evidence computation and posterior estimation — great for multi-modal parameter spaces.
