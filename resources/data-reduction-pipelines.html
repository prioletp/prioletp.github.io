Data Reduction Pipelines

⚙️

A curated list of official and community-maintained pipelines for reducing data from interferometric instruments. These tools take you from raw detector frames to calibrated interferometric observables.

VLTI Instrument Pipelines (ESO)

⚙️
MATISSE Pipeline
ESO's official data reduction pipeline for the MATISSE instrument (L, M, N bands). Produces calibrated visibilities, closure phases, and differential phases.
↗ ⚙️
GRAVITY Pipeline
ESO's official data reduction pipeline for the GRAVITY instrument (K band). Handles fringe tracking, spectral calibration and visibility extraction.
↗ ⚙️
PIONIER Pipeline (pndrs)
The PIONIER data reduction software for H-band interferometric observations at the VLTI.
↗

Community & Third-Party Tools

mat_tools
Python utilities for post-processing and analysing MATISSE pipeline outputs — averaging, calibration and diagnostic plots.
↗ 🔄
EsoReflex
ESO's graphical workflow environment for running instrument pipelines — useful for beginners who want a visual interface.
↗ 📂
Gasgano
ESO's data file organiser for browsing, classifying and organising raw FITS files before pipeline reduction.
