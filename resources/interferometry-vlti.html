Interferometry & VLTI

🔭

Essential documentation and reference material for working with the Very Large Telescope Interferometer and understanding optical/infrared interferometry.

VLTI Documentation

🔭
VLTI Overview (ESO)
Official ESO page for the Very Large Telescope Interferometer — instrument descriptions, capabilities and technical specifications.
↗ 📡
MATISSE Instrument Page
Documentation for the Mid-infrared And Thermal-infrared Spectrograph and imager for interferometry — L, M and N band capabilities.
↗ 📡
GRAVITY / GRAVITY+ Instrument Page
Documentation for GRAVITY, the K-band beam combiner with dual-field capability, fringe tracking and astrometric mode.
↗ 📡
PIONIER Instrument Page
Documentation for the H-band 4-telescope beam combiner, widely used for surveying circumstellar environments.
↗

Data Format & Standards

📄
OIFITS Standard
The Optical Interferometry FITS data format standard (OI-FITS v2) for exchanging calibrated visibilities, closure phases and wavelength tables.
↗ 📄
FITS Standard (NASA)
The Flexible Image Transport System standard — the underlying file format used throughout astronomy and interferometry.
↗

Other Interferometric Facilities

🏔️
CHARA Array
The Center for High Angular Resolution Astronomy array at Mount Wilson — the longest optical/IR baselines in the world (up to 330 m).
↗ 🏔️
LBTI (Large Binocular Telescope)
The Large Binocular Telescope Interferometer — nulling interferometry for exozodiacal dust detection (HOSTS survey).
