Interferometric Modelling Tools

📐

A collection of Python scripts and modules for modelling and fitting interferometric observables — visibilities, closure phases, and differential phases — with parametric models of circumstellar environments. These tools were developed throughout my PhD for analysing VLTI data of exozodiacal dust around nearby stars.

Features

• Parametric models — uniform disks, thin rings, Gaussian rings, power-law radial profiles, and binary/point-source models
• Chromatic models — wavelength-dependent visibility fitting for spectrally dispersed instruments (MATISSE, GRAVITY)
• Correlation matrices — proper handling of correlated errors in interferometric data, including empirical correlation matrix estimation
• MCMC fitting — Bayesian parameter estimation using emcee with built-in convergence diagnostics and corner plots
• Nested sampling — model comparison via Bayesian evidence using dynesty
• OIFITS I/O — read and write OIFITS files, extract baselines, wavelengths, and observables

Installation

Requires Python 3.9+ with numpy, scipy, matplotlib, astropy, emcee, corner, and optionally dynesty. 

# Clone the repository
git clone https://github.com/prioletp/interf-modelling.git
cd interf-modelling

# Install dependencies
pip install -r requirements.txt

# Install the package
pip install -e .

Quick Start

Fit a thin ring model to an OIFITS file: 

from interf_modelling import OIData, ThinRingModel, MCMCFitter

# Load data
data = OIData.from_oifits('my_observation.fits')

# Define model and priors
model = ThinRingModel()
priors = {
    'f_ring': (0.001, 0.05),   # ring-to-star flux ratio
    'r_ring': (0.5, 5.0),      # ring radius in au
    'inc': (0, 90),             # inclination in degrees
    'pa': (0, 180),             # position angle
}

# Run MCMC
fitter = MCMCFitter(data, model, priors, use_corr_matrix=True)
result = fitter.run(n_walkers=32, n_steps=5000, burn_in=1000)

# Plot results
result.corner_plot()
result.best_fit_plot()
