pyGRaTeR

💻

pyGRaTeR (Python GRains Temperature and Radiation) is an open-source radiative transfer code designed to model the thermal emission and scattering properties of optically thin debris disks. It was developed as part of my PhD research on exozodiacal dust at IPAG.

Features

pyGRaTeR computes the temperature and spectral energy distribution (SED) of dust grains in optically thin environments. Key features include:

• Mie theory — full Mie scattering and absorption cross-sections for spherical grains of arbitrary composition and size
• Multi-grain modelling — handles grain size distributions with customisable power-law indices, minimum/maximum sizes, and blowout size computation
• Dust compositions — supports silicates, carbon, iron, and custom optical constants from laboratory measurements
• N-body inputs — can ingest spatial distributions from N-body simulations to produce synthetic observables
• Interferometric observables — generates synthetic visibilities and closure phases for comparison with VLTI data
• SED fitting — built-in routines for fitting photometric data points and comparing with models

Installation

pyGRaTeR requires Python 3.9+ and the following dependencies: numpy, scipy, matplotlib, astropy, and miepython.

Clone the repository and install in development mode: 

# Clone the repository
git clone https://github.com/prioletp/pyGRaTeR.git
cd pyGRaTeR

# Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate

# Install dependencies and the package
pip install -e .

Quick Start

A basic example to compute the temperature and SED of silicate grains around a solar-type star: 

import pygrater as pg

# Define the star
star = pg.Star(T_eff=5778, R_star=1.0, L_star=1.0)

# Define the dust grain properties
dust = pg.DustGrain(composition='silicate', a_min=0.1, a_max=1000, q=-3.5)

# Compute grain temperatures at a range of distances
temps = pg.compute_temperatures(star, dust, r_min=0.1, r_max=10, n_radii=100)

# Generate the SED
sed = pg.compute_sed(star, dust, temps, wavelengths=pg.default_wavelengths())

# Plot
sed.plot()
