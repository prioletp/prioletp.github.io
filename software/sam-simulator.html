SAM Simulator

🎯

SAM Simulator is a simulation pipeline for the Sparse Aperture Masking (SAM) mode of VLT/SPHERE. It was developed during my M1 internship at the Laboratoire d'Astrophysique de Marseille (LAM) to characterise point-source detection limits and calibration strategies for the SAM observing mode.

What is Sparse Aperture Masking?

SAM is a high-angular-resolution technique where a non-redundant mask is placed in the telescope pupil, converting the full aperture into an interferometric array. The resulting interference pattern encodes closure phases — observables that are immune to atmospheric phase errors — enabling the detection of faint companions at separations well below the classical diffraction limit.

Features

• Mask geometry — simulates the 7-hole SAM mask of SPHERE/IRDIS with accurate sub-aperture positions
• Atmospheric model — includes Kolmogorov turbulence, Strehl ratio variations, and AO correction effects
• Detection maps — computes contrast curves and detection limits as a function of angular separation
• Binary injection — injects simulated companions at arbitrary separations, position angles, and contrast ratios
• Closure phase analysis — extracts and analyses closure phases from simulated interferograms
• Calibration strategies — tests different calibrator selection and observation sequence strategies

Installation

Requires Python 3.8+ with numpy, scipy, matplotlib, and astropy. 

# Clone the repository
git clone https://github.com/prioletp/sam-simulator.git
cd sam-simulator

# Install dependencies
pip install -r requirements.txt

# Run the simulator
python sam_simulator.py --config examples/default_config.yaml

Quick Start

Generate a detection limit map for a bright star with SPHERE/SAM: 

from sam_simulator import SAMMask, Observation, DetectionMap

# Set up the mask and observation
mask = SAMMask(instrument='SPHERE-IRDIS')
obs = Observation(
    star_mag=5.0,
    band='H',
    dit=0.2,
    n_frames=1000,
    strehl=0.85
)

# Compute detection limits
det = DetectionMap(mask, obs)
det.compute(n_simulations=500)

# Plot the 5-sigma contrast curve
det.plot_contrast_curve(sigma=5)
