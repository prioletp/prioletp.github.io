Jupyter Teaching

Teaching Notebooks

📓

A collection of interactive Jupyter notebooks created for the optics and polarisation practical sessions I taught at Université Grenoble Alpes in 2024. These notebooks are designed for undergraduate physics students (L3/M1 level) and combine theory, interactive visualisations, and guided exercises.

Available Notebooks

The collection covers the following topics:

• Diffraction & Interference — single slit, double slit, and N-slit diffraction patterns with interactive parameter exploration
• Fourier Optics — spatial frequencies, optical transfer function, and the link between diffraction and Fourier transforms
• Polarisation — Jones calculus, Stokes parameters, Müller matrices, and polarimetric measurements
• Interferometry Basics — two-beam interference, visibility, coherence, and the van Cittert–Zernike theorem
• Optical Instruments — telescopes, spectrographs, and interferometers — resolution limits and design trade-offs

Installation

The notebooks require Python 3.8+ with jupyter, numpy, matplotlib, and ipywidgets for interactive plots. 

# Clone the repository
git clone https://github.com/prioletp/teaching-notebooks.git
cd teaching-notebooks

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook

Alternatively, you can run the notebooks directly in your browser using Binder (link in the repository README) — no local installation required.

Usage

Each notebook is self-contained and includes:

• A brief theory introduction with equations and diagrams
• Interactive widgets to explore how changing parameters affects the physics
• Guided exercises with empty code cells for students to fill in
• Solutions in separate companion notebooks

The notebooks are written in French (as used in the course), but the code and comments are in English for broader accessibility.

Source Code
